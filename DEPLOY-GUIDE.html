Aera · Deploy Guide

Go Live in
3 Steps

GitHub → Vercel → Your Domain · Free

✦

Your file package

You have 4 files. These go into one folder on your computer — call it aera-app.

aera-app/
index.html ← landing page
app.html ← Frequency Index™ app
manifest.json ← makes it installable
sw.js ← offline support

App icon: You'll also need two PNG icons — 192×192px and 512×512px — saved as icons/icon-192.png and icons/icon-512.png inside the folder. Use Canva or any image tool to make a simple square logo on a dark background.

1

Put your files on GitHub

GitHub stores your code and connects to Vercel for deployment. It's free.

1. Go to github.com and create a free account if you don't have one.

2. Click the + icon → New repository

3. Name it aera-app → set to Public → click Create repository

4. On the next screen, click uploading an existing file

5. Drag and drop all your files (index.html, app.html, manifest.json, sw.js, and the icons folder) into the upload area.

6. Scroll down → click Commit changes

✓ Done when: You can see all your files listed on your GitHub repository page.

2

Deploy to Vercel (free hosting)

Vercel turns your GitHub files into a live website instantly. No configuration needed — it just works with plain HTML files.

1. Go to vercel.com → click Sign Up → sign in with your GitHub account.

2. Click Add New → Project

3. Find your aera-app repository and click Import

4. Leave all settings as default — Vercel will auto-detect it's a static site.

5. Click Deploy

6. Wait ~30 seconds. Vercel gives you a free URL like aera-app.vercel.app

✓ Done when: You see a green "Congratulations" screen and can open your live URL.

3

Connect your domain

You already have a domain. This connects it to your Vercel deployment so members go to your real URL.

1. In your Vercel project, go to Settings → Domains

2. Type in your domain (e.g. aeraelevation.com) → click Add

3. Vercel will show you two DNS records to add. Copy them.

4. Go to wherever you registered your domain (GoDaddy, Namecheap, Squarespace, etc.) → find DNS Settings

5. Add the records Vercel gave you — usually one A record and one CNAME record

6. Wait up to 24 hours (usually under 1 hour) for DNS to update

7. Vercel auto-installs a free SSL certificate — your site will be https:// automatically

✓ Done when: Visiting your domain loads the Aera landing page.

✦

How members "install" the app

Once your site is live, members can add it to their home screen — it looks and feels like a downloaded app with no App Store needed.

iPhone: Open the site in Safari → tap the Share button (box with arrow) → scroll down → tap Add to Home Screen → tap Add.

Android: Open in Chrome → tap the three-dot menu → tap Add to Home screen → tap Add.

The app will appear on their home screen with the Aera icon, open full-screen with no browser bar, and work offline.

✦

How to update the app going forward

Every time you want to change something, just replace the file on GitHub. Vercel auto-redeploys in under 30 seconds.

Go to your GitHub repo → click the file you want to update → click the pencil ✏️ icon to edit → make changes → click Commit changes. Done. Vercel detects the change and redeploys automatically.
